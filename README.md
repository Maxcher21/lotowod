# ЛОТОВОД

Бот для лотереи на голосе. Находит посты @golos.loto, голосует за них и оставляет коментарий со случайными цифрами. 

## Зависимости

* Требууется нода 7 и выше
* npm модуль steem

## Запуск

```
node . -v 
```

Где 

-v - отладочные сообщения
.vv - отдадочные и трассировочные сообщения
